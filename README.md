# firebase_chatapp

Flutter ile geliştirilmiş, kullanıcı kayıt giriş işlemleri ve mesajlaşma depolaması firebase tarafından kaydedilip kontrol edilen mobil chat uygulamasıdır.

## Proje içeriği
- Firebase auth
- Firebase firestore
- Firebase storage
